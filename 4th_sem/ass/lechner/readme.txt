In order to edit the document machinelearning.pdf, you have to edit the latex files.

for instance:

	vi MIT_SAD/4th_sem/ass/lechner/questions/machinelearning.tex

After you edit the latex file, you have to generate the machinelearning.pdf. Make sure you are in the right directory.

cd MIT_SAD/4th_sem/ass/lechner/questions

In this directory you have to perform the following command in order to generate the pdf.

pdflatex machinelearning.tex


Please do not commit, if you cant create the machninelearning.pdf file.
